# Adobe Acrobat Desktop Automation POC

## Overview

This project demonstrates desktop automation of Adobe Acrobat using:

* Java
* WinAppDriver
* Appium Java Client
* TestNG

## Features Automated

* Open PDF
* Zoom In
* Scroll Pages
* Search Text in PDF

## Framework Design

The framework follows Page Object Model (POM).

Project Layers:

* Base Layer
* Page Layer
* Utility Layer
* Test Layer

## Tools Used

* Eclipse IDE
* WinAppDriver
* Adobe Acrobat Reader
* Maven

## How to Execute

1. Start WinAppDriver
2. Open Adobe Acrobat manually
3. Run `testng.xml`

## Test Cases

* Verify PDF Open
* Verify Zoom In
* Verify Scroll Down
* Verify Search Text
